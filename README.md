# TaskPlanet Social App 🚀

A full-stack MERN social media application where users can create posts, like, and comment in a modern social feed.

---

## 🌐 Live Demo
https://taskplanet-frontend-three.vercel.app

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- CSS (Responsive UI)

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

---

## ✨ Features

- 🔐 User Authentication (Signup/Login)
- 📝 Create posts (Text + Image)
- ❤️ Like posts
- 💬 Comment on posts
- 🔍 Search functionality
- 📄 Pagination (Load more posts)
- 📱 Fully responsive UI
